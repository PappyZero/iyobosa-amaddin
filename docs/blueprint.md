# **App Name**: MAJIDLABS

## Core Features:

- Code Explainer: AI-powered tool that analyzes and explains code snippets from projects, providing insights into functionality and architecture.
- Skills Showcase: Dynamically display categorized skills with proficiency levels, emphasizing frontend and blockchain expertise.
- Project Gallery: Present featured projects with descriptions, tech stacks, and links to live demos or GitHub repositories.
- Interactive Timeline: Visually represent the experience and education using an interactive timeline.
- Blog Highlight: Feature the latest blog posts and articles with links to Medium or other publishing platforms.
- Contact Form: Simple, direct contact form for inquiries.
- ATS-Optimized Structure: Ensure the portfolio structure adheres to ATS best practices for readability and keyword matching.

## Style Guidelines:

- Background: #0F172A (deep slate)
- Primary Accent: Cyan / Teal
- Secondary Accent: Soft Blue
- Text: Slate-200 / Slate-400
- Headings: Inter / Poppins
- Body: Inter / Source Sans Pro
- Code: JetBrains Mono (optional)
- Code font: 'Source Code Pro' for displaying code snippets.
- Clean and sectioned layout, with a clear navigation menu, and mobile responsive design.
- Simple, professional icons to represent skills and links, prioritizing readability and minimalism.
